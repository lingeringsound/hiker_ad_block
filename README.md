### 订阅地址
#### 复制到海阔视界 或者 嗅觉浏览器

```
{"urlV2":"https://raw.fgit.ml/lingeringsound/hiker_ad_block/main/css.prop","domBlockRuleUrl":"https://raw.fgit.ml/lingeringsound/hiker_ad_block/main/domain.prop"}
```
